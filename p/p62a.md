#Problem 62a

Count internal nodes (those that have non-empty subtrees) of a binary tree.

##Example
```elm
tree = Tree 'a' (Tree 'b' Empty (Tree 'c' Empty Empty))
                 (Tree 'd' Empty (Tree 'e' Empty Empty))

countInternals tree == 3
```
