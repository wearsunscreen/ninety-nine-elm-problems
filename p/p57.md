# Problem 57

Build a binary tree from a list.

##Example 
```elm
toTree [1..4] == Node 1 
                  (Node 2 Empty (Node 4 Empty Empty)) 
                  (Node 3 Empty Empty)
```

    = Empty
    | Node a (Tree a) (Tree a)

        , testBalance (balancedTree 3) 3 == True

        , testBalance (balancedTree 5) 5 == True

        , testBalance (balancedTree 73) 73 == True
        , testBalance (balancedTree 5000) 5000 == True

testBalance : Tree a -> Int -> Bool
testBalance tree n = 
    case tree of 
        Empty -> 
            n == 0
        
        Node node left right ->
            List.all ((==) True)
              [ (abs (count left) - count (right)) < 2
              , count tree == n
              ]
    
    
-- count number of Nodes in a Tree    
count : Tree a -> Int
count tree = 
    case tree of 
        Empty -> 0
        
        Node n left right ->
            1 + (count left) + (count right)
