# Problem 68c

Write a function to traverse a tree and return the values in a list in post-order.


## Example
```elm
tree67 = 
    Node 'a'
        (Node 'b'
            (Node 'd' Empty Empty)
            (Node 'e' Empty Empty))
        (Node 'c' 
            Empty 
            (Node 'f' 
                (Node 'g' Empty Empty) 
                Empty))

postorder tree67 == ['d', 'e', 'b', 'g', 'f', 'c', 'a']  
```
