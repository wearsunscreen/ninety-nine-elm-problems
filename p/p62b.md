# Problem 62b

Extract the internal nodes (those that have non-empty subtrees) of a binary tree into a list.

##Example
```elm
tree = Tree 'a' (Tree 'b' Empty (Tree 'c' Empty Empty))
                 (Tree 'd' Empty (Tree 'e' Empty Empty))

getInternals tree == ['a', 'b', 'd']
```
